## 登录接口

#### 1. URL

api/app/users/login

#### 2. 请求头

- account,必须有,用户名|邮箱
- password,必须有,密码的md5值
- X-Deviceid,必须有
- token,必须有,签名md5(account+password+X-Deviceid)

#### 3. 请求方式

POST

#### 4. 返回值示例

成功:
```json
{
  "code": 0,
  "msg": "接口调用成功",
  "data": {
      "token": "JKLWESJKDS83"
  }
}
```

失败:
```json
{
  "code": 500,
  "msg": "服务器内部错误,请稍后再试",
  "data": null
}
```

#### 5. 错误码code说明

- code: 0, 操作成功
- code: 2000, 账号或密码错误
- code: 2001, 请求错误(token错误)
- code: 2002, 未知错误,请稍后再试
